# Données Boursières BRVM (Côte d'Ivoire)

Ce projet récupère automatiquement les données du tableau du site Portail Economie Ivoirienne.

## Fichiers
- `collect_brvm_data.py` : Script qui extrait les données
- `data/brvm_market_data.csv` : Fichier CSV contenant les données

## Mise à jour automatique
Les données sont mises à jour tous les jours à 9h (heure d'Abidjan).
